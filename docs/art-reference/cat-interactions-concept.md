# Cat interaction concept prompt

- Mode: built-in imagegen
- Use case: `stylized-concept`
- Asset type: visual reference sheet for an existing mobile pixel-art apartment simulation game

## Final prompt

Create a clean five-panel sprite reference sheet showing two small house cats interacting in five clearly different ways: playful chase, affectionate grooming, curled-up shared nap, tense territorial face-off, and cooperative mischievous play with a tipped cardboard box.

Use the same flat muted lavender floor-tile backdrop in every vignette, separated by generous empty gutters. Show two compact, expressive pixel-art cats—one orange tabby and one dark charcoal cat—in crisp hand-authored 16-bit pixel art from a top-down three-quarter view, consistent with a cozy management simulation. Use a deliberately limited palette, hard pixel edges, consistent scale and camera angle, warm playful lighting, and silhouettes that remain readable at tiny mobile size.

No words, labels, UI chrome, gradients, anti-aliased edges, watermark, photorealism, smooth vector art, painterly texture, 3D rendering, excessive detail, or humans. Keep every cat fully visible and avoid silhouette overlap that makes them look like one animal.
