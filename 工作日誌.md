# 《房東監視中》開發工作日誌

> 更新日期:2026-07-06
> 一款「異步掛機觀察型」直式手機模擬遊戲:玩家是房東,透過俯視娃娃屋觀察租客生活、佈置家具、招租收租,劇情由 AI 依歷史生成(AI 部分尚未接線)。

---

## 一、目前完成度總覽

「**觀察 + 房東經營**」的骨架大致完成且可玩;缺的是「**AI 敘事靈魂**」與「**數值後果/平衡**」。目前所有功能**完全離線、無需後端**即可跑。

---

## 二、已完成的系統

### A. 核心資料與美術
- **核心資料結構** `src/types.ts` — `Tenant` / `Furniture` / `AIEventResponse`、15 種 visual_state、房間屬性軸。
- **種子資料** `data/tenants.json`(陳家豪、林小婕)、`data/furniture.json`。
- **AI 提示詞範本** `prompts/ai-engine-prompt.md` — 三層架構 + JSON Schema + 後端守則(之後接 AI 直接用)。
- **像素美術系統** `src/pixel/sprites.ts`、`scene.ts` — 俯視 3/4、5 階色階、角色站/坐/躺/走 sprite、動態租客配色池(4 種)。

### B. 樓層地圖(tile 網格)
- `src/floor/map.ts` — 16×33 網格:301/302/303/304 四間套房 + 中央交誼廳 + 廁所浴室(乾濕分離)+ 洗衣晾衣間 + 明顯大門,可上下捲動。
- `src/floor/floorScene.ts` + `src/components/FloorMap.vue` — 樓層渲染、房間標籤、未讀徽章、點擊進房/擺放。
- 磁磚/木地板區分、牆體 3/4 厚度、門與門檻。

### C. 家具目錄(單一資料來源)
- `src/furniture/catalog.ts` — 約 35 件家具,每件含價格/佔位/屬性/適合性格/解鎖狀態/社交點/敘事鉤子。新增家具只需加一筆資料。
- `src/furniture/render.ts` — 專屬繪圖 + 「零件清單」資料化渲染器。
- `src/sim/placements.ts` — **動態**家具狀態(可買/擺/移除)、房間屬性計算、找空位、擺放判定。

### D. 走動與尋路
- `src/floor/pathfind.ts` — BFS 尋路 + 障礙格(家具佔用格自動不可走)+ 版本快取。
- `src/floor/agents.ts` — 狀態驅動的行為體:讀取每位租客當前活動目標 → 走過去 → 停下;外出時隱藏;不再隨機漫遊。

### E. 異步掛機模擬
- `src/sim/clock.ts` — 8× 掛機時鐘(現實 1 天 = 遊戲 8 天),關掉再開自動補進度。
- `src/sim/routine.ts` — 每位租客 24 小時作息表 + 招租原型作息(上班族/學生/自由接案)+ 家具角色反查(只用自房或共用區)。
- `src/sim/generate.ts` — **內容生成接縫**:目前是模板句,**這是之後接 Claude 的唯一插入點**。
- `src/store.ts` — 每小時 tick、作息+偏離、未讀模型、存檔(v2 localStorage)、快轉。

### F. 房東經營循環(經濟)
- **家具商店** `src/components/FurnitureShop.vue` — 選家具 → 回地圖點格擺放(壓牆/重疊/跨房間會擋)→ 扣款。
- **收租** — 每遊戲日自動入帳,依付租能力 × 好感。
- **房間屬性** — 家具累積屬性,房間細看可見。
- **空房招租** `src/sim/recruit.ts` + `src/components/RecruitPanel.vue` — 依裝潢屬性產生應徵者、算契合度(★)、選一位入住 → 有作息、繳租。
- 動態房間佔用(`state.occupancy`)、動態租客配色。

### G. UI
- `src/App.vue` — 樓層總覽 ↔ 房間細看並存、快轉鈕、商店/招租/擺放橫幅、抉擇彈窗。
- `src/components/LogFeed.vue` — 未讀「自上次查看」分隔線 + NEW 高亮。
- `DecisionModal.vue`、`PixelDollhouse.vue`、`style.css`。

### H. 自我檢測 / 開發工具(`scripts/`)
- `sim-trace.ts` — **無頭模擬追蹤器**:跑真邏輯,印出誰走到哪、有無卡住,自動健全性檢查(改完程式自我驗證用)。
- `routine-test.ts`、`shop-test.ts`、`recruit-test.ts`、`place-test.ts`、`newtenant-test.ts`、`pf-test.ts`。
- `render-floor.ts`、`render-preview.ts` — 離線渲染 PNG 檢視美術。

## 六、如何執行 / 驗證
```bash
# 開發伺服器(瀏覽器開 http://localhost:5173)
npm run dev

# 無頭自我檢測(改完程式驗證走位/經濟/招租)
npx tsx scripts/sim-trace.ts 24      # 模擬 24 遊戲小時,檢查走位
npx tsx scripts/recruit-test.ts      # 招租入住流程
npx tsx scripts/shop-test.ts         # 買家具

# 離線渲染樓層 PNG
npx tsx scripts/render-floor.ts out.png
```
